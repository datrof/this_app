Программа для онлайн-мониторинга данных в Ловозере

Перед работой должен быть задан файл для flat-fielding в формате ff*now.txt, такой файл может быть только один.

Для запуска программы набрать в командной строке:
./run_this_online.sh /usr/local/MATLAB/MATLAB_Runtime/v97 /home/lovozero/paip
, где
/usr/local/MATLAB/MATLAB_Runtime/v97 - место расположения программы Matlab_Runtime
/home/lovozero/paip - директория с .dat-файлами Ловозера
